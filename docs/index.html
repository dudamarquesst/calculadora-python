Calculadora em Python

Projeto desenvolvido para o Bootcamp I - Engenharia de Software

Sobre o Projeto

Este projeto é uma calculadora simples desenvolvida em Python. Ele faz parte do desafio de versionamento e portfólio pessoal para o Bootcamp I da faculdade.

Funcionalidades

  • Adição, subtração, multiplicação e divisão
  • Interface simples e intuitiva via terminal
  • Código organizado com boas práticas de versionamento
